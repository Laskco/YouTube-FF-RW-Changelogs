Version 4.3.4

November 16th, 2025

A minor UI Tweak to refine the positioning of the forward and rewind buttons.

UI Changes:

  • Improved Button Placement: The Fast-Forward and Rewind buttons have been repositioned for a better fit within the player's control bar.
  • Tighter Alignment (Left Position): When the buttons are set to the left side, they are now shifted to sit perfectly flush next to the time display, removing the previous gap.
  • No Change for Right Position: This adjustment only affects the left-side layout. The button placement on the right side of the player remains unchanged.

Version 4.3.2

November 10th, 2025

Minor bug fix release improving user experience in the settings interface.

Bug Fixes:

  • Fixed advanced settings panel closing when using the scrollbar in settings view
    • Changed click-away detection logic to only trigger when clicking on other settings cards
    • Scrollbar interactions and empty space clicks no longer inadvertently close the panel

Version 4.3.1

November 5th, 2025

Focused on stability, performance, and reliability improvements across all systems.

Major Changes

  • Fixed critical popup loading issue.
  • Resolved race condition where background scripts continued running after disable.

Performance

  • Reduced CPU usage by optimizing hold-to-seek logic.
  • Adjusted update delays for smoother playback.

Version 4.3.0

November 1, 2025

What's New

  • Full Support for New YouTube UI: The extension now works seamlessly with YouTube's updated player design that's rolling out to all users.
  • Better Button Icons: Replaced old PNG icons with crisp SVG graphics that look perfect at any size.
  • Improved Button Injection: Fixed several bugs where buttons wouldn't show up or would appear in the wrong place.
  • Smoother Animations: Button appearance and transitions are now more fluid and reliable.

Version 4.2.8

August 27, 2025

Improvements

  • Button Click Feedback: All buttons now have a more dynamic and responsive press animation.
  • Smoother Button Loading: The fast-forward and rewind buttons now load with a consistently smooth animation.
  • Code Efficiency: Refined the code for better performance, especially in video player detection and settings validation.

Bug Fixes

  • Advanced Settings Panel: Fixed an issue that caused the advanced settings panel to close when its border or scrollbar was clicked.
  • Button Animation: Corrected an issue where the button loading animation would sometimes lag or appear inconsistent.

Version 4.2.7

August 19, 2025

Bug Fixes

  • Fixed CSS typo affecting light theme button hovers.
  • Fixed accessibility issue where hidden tooltips were read by screen readers.
  • Fixed skip indicator overlay blocking YouTube menus/captions.

Performance

  • Reduced DOM lookups by caching elements.
  • Debounced settings saves to reduce storage writes.

Features

  • New Advanced Setting: Added a "Progress Bar Update Delay" setting.
  • New Advanced Setting: Added a "Navigation Init Delay" setting.
