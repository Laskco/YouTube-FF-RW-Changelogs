4.2.6

Technical Updates & Fixes

This update focuses on major behind-the-scenes improvements to make the extension more stable, reliable, and future-proof.

Key Improvements

  • Complete Backend Refactor: The extension's code has been rewritten for improved stability and efficiency, making it less likely to break when YouTube updates its site.
  • Smarter Error Handling: Improved error detection and recovery mean the extension can now handle unexpected issues more gracefully without crashing.

Bug Fixes

  • Continuous Seek Fix: Fixed a bug where holding a hotkey and switching tabs would cause the video to seek indefinitely.

Future Plans

  • Give users the ability to change the Fast Forward & Rewind Buttons to any value they want.
  • Allow users to create their own skipping indicators for the keyboard shortcuts, with several pre-made designs available.

Known Issues

Massive skips when Buffering Protection is off

Issue: Disabling "Buffering Protection" can cause large, erratic jumps while skipping when the video is buffering. This is a limitation of YouTube's player when it's under heavy load.

Workaround: If you experience this, try lowering the video quality or go to the extension's settings and increase the "Seek Throttle (ms)" to 150-250ms. This gives the player more time to catch up.

Potential Incompatibility with New YouTube UI

Warning: YouTube is testing a new player UI. As this new design has not been rolled out to us for testing, the extension's compatibility with it is unknown. If you encounter the new UI, please report any issues on GitHub so we can address them.
