Known Issues

Massive skips when Buffering Protection is off

Issue: Disabling "Buffering Protection" can cause large, erratic jumps when the video is buffering, especially in high quality.

Potential Incompatibility with New YouTube UI

Warning: YouTube has been testing a new player UI, similar to the Google Drive player. Unfortunately, I have no idea how my extension will react to this change if they push it globally. I have been unlucky and still haven't gotten the new UI yet, so if you have my extension installed and you get the new UI, please let me know on GitHub if the extension completely breaks or not.
